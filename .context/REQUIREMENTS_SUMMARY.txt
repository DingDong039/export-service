CRITICAL QUESTIONS
================================================================================

ตอบคำถามเหล่านี้เพื่อให้ได้ requirements ที่ชัดเจน:

1. Primary use case? 
   □ One-time reports  □ Scheduled exports  □ User downloads  □ API integration

2. Data volume?
   □ < 1,000 rows  □ 1k-100k rows  □ 100k+ rows  □ Real-time streaming

3. Deployment?
   □ Standalone  □ Integration with existing app  □ Microservice  □ Serverless

4. User base?
   □ < 100 users  □ 100-1k users  □ 1k+ users

5. Budget?
   □ Open source only  □ Small budget  □ Enterprise budget

6. Timeline?
   □ MVP in 1-2 weeks  □ Full in 1 month  □ Long-term project
